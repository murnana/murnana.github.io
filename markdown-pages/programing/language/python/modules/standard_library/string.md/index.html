string

書式指定文字列

'{0}, {1}, {2}'.format('a', 'b', 'c')
'Coordinates: {latitude}, {longitude}'.format(latitude='37.24N', longitude='-115.81W')

フォーマット済み文字列リテラル

f"{'a'}, {'b'}, {'c'}"

latitude = '37.24N'
longitude='-115.81W'
f"Coordinates: {latitude}, {longitude}"
